# QuoteFlow

QuoteFlow — калькулятор сметы и генератор коммерческого предложения для малого бизнеса, экспертов и небольших команд.

Проект является собственным демонстрационным продуктом Stalar Vision. Он создается как честный публичный пример подхода к проектированию и разработке продукта, а не как клиентский кейс. В проекте не используются вымышленные клиенты, отзывы, пользовательская статистика или неподтвержденные коммерческие результаты.

## Практическая задача

QuoteFlow должен помочь подготовить расчет по позициям, применить скидки и налог, сохранить черновик в браузере и сформировать понятное коммерческое предложение. MVP ориентирован на демонстрационные и synthetic/demo data, а не на реальные персональные, коммерческие или конфиденциальные данные.

## Планируемые возможности MVP

- расчет сметы по позициям;
- скидка на отдельную позицию и общая скидка;
- налог после общей скидки;
- хранение черновиков в `localStorage`;
- JSON import/export на frontend;
- CSV export на frontend;
- backend preview для повторной валидации и пересчета;
- PDF generation на backend на следующем этапе развития;
- единый расчетный contract между frontend и backend.

Все возможности в этом разделе являются планируемыми, а не реализованными.

## Предполагаемый стек

- Frontend: React, TypeScript, Vite, Tailwind CSS, Vitest, Testing Library.
- Backend: Python, FastAPI, Pydantic, pytest.
- Draft storage: только браузерный `localStorage`.
- Server storage: в MVP не используется.

## Архитектурный принцип

Frontend отвечает за UI, редактирование черновиков и локальный пользовательский опыт. Backend остается stateless, повторно валидирует входные данные и самостоятельно пересчитывает totals перед формированием preview или PDF. Черновики не синхронизируются с backend.

## Статус

Текущий статус проекта: `architecture and documentation stage`.

На текущем этапе нет production deployment, нет работающего frontend/backend-кода, нет установленных зависимостей и нет лицензии. Лицензия будет выбрана отдельно на более позднем этапе.

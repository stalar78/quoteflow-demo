## Назначение

QuoteFlow задуман как калькулятор сметы и генератор коммерческого предложения для малого бизнеса, экспертов и небольших команд. Продукт одновременно решает понятную прикладную задачу и демонстрирует компетенции Stalar Vision в бизнес-калькуляторах, web-интерфейсах, REST API, обработке структурированных данных, PDF generation и тестировании бизнес-логики.

## Принятые решения

- Рабочее название: `QuoteFlow`.
- Репозиторий: `stalar78/quoteflow-demo`.
- Репозиторий пока приватный; публичная публикация требует security/publication review.
- Лицензия пока не выбрана и не добавляется без отдельного решения.
- Frontend: React, TypeScript, Vite, Tailwind CSS, Vitest, Testing Library.
- Backend: Python, FastAPI, Pydantic, pytest.
- Черновики хранятся только в браузере через `localStorage`.
- Backend остаётся stateless.
- Server database в MVP отсутствует.
- Платежи, CRM, реальные webhook/email/Telegram-интеграции, AI и пользовательские аккаунты не входят в MVP.
- Production deployment требует отдельного review.

## Ограничение данных

Публичное demo не предназначено для реальных персональных, коммерческих или конфиденциальных данных. Для разработки, тестов, screenshots и демонстрации используются только synthetic/demo data.

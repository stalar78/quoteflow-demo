# 00. Контекст проекта

QuoteFlow — самостоятельный проект в каталоге `QuoteFlow`. Он не является частью сайта Stalar Vision и не должен изменять файлы основного сайта.

## Назначение

QuoteFlow задуман как калькулятор сметы и генератор коммерческого предложения для малого бизнеса, экспертов и небольших команд. Продукт должен показывать, как Stalar Vision проектирует небольшое прикладное решение: от ограничений и расчетной модели до API contract и security boundaries.

## Владение и позиционирование

Владелец проекта:

- Ларин Станислав;
- Stalar Vision;
- независимый разработчик / ИП.

QuoteFlow является собственным демонстрационным продуктом Stalar Vision. Его нельзя выдавать за клиентский проект. В публичных материалах допустимо описывать его как собственный продуктовый пример, demo или case study без утверждений о внешнем заказчике.

## Запрещенные утверждения

В документации и будущих публичных материалах нельзя придумывать:

- клиентов;
- отзывы;
- пользовательскую статистику;
- коммерческие результаты;
- экономию времени;
- количество сформированных документов;
- uptime;
- сроки окупаемости;
- цены разработки;
- любые неподтвержденные показатели.

## Принятые решения

- Рабочее название: `QuoteFlow`.
- Предварительное имя будущего GitHub repository: `stalar78/quoteflow-demo`.
- GitHub repository на Этапе 1 не создается.
- Внешняя публикация и production deployment на Этапе 1 не выполняются.
- Лицензия на Этапе 1 не добавляется.
- Drafts хранятся только в браузере через `localStorage`.
- Backend остается stateless.
- Server database в MVP не используется.
- Реальные платежи, CRM, webhook/email/Telegram-интеграции и AI не входят в MVP.

## Ограничение данных

Публичное demo не предназначено для реальных персональных, коммерческих или конфиденциальных данных. Для разработки, проверки и демонстрации должны использоваться только synthetic/demo data.
